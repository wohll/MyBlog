# 技术栈：
#### 后端：Springboot+Mybatis
#### 前端：Thymeleaf+Layui+jQuery,集成了Editor.md编辑器
#### 数据库：MySQL
# 个人博客项目介绍：
##### 1. 登录模块: "/admin",未登录之前只能访问首页和查看文章，登录之后进入后台
##### 2. 后台: 支持编辑文章，发表文章，删除文章，修改文章
##### 3. 博客首页，个人的信息和文章的展示
##### 4. 数据库表结构：两张表，tb_user和tb_blog
# 运行项目：
##### 1. 使用sql脚本创建两张表
##### 2. 在配置文件中写好你自己的数据库用户名和密码，还有数据库名称点击运行即可
